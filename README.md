## 📌 Sobre o projeto

Este é um sistema web simples desenvolvido em **PHP puro**, com o objetivo de realizar o controle básico de finanças.

O sistema permite cadastrar receitas e despesas, calcular saldo automaticamente e visualizar um histórico das transações.

---

## 🚀 Funcionalidades

* 🔐 Login de acesso
* ➕ Cadastro de receitas e despesas
* 📊 Cálculo automático do saldo
* 📋 Histórico de transações
* 🎨 Interface estilizada com CSS

---

## 🛠️ Tecnologias utilizadas

* PHP
* HTML
* CSS

---

## ⚙️ Como executar o projeto

### 1. Clone ou baixe o repositório

```bash
git clone https://github.com/seu-usuario/sistema-financeiro.git
```

### 2. Coloque na pasta do servidor local

* XAMPP → `htdocs`
* WAMP → `www`

### 3. Acesse no navegador

```
http://localhost/sistema-financeiro/index.php
```
